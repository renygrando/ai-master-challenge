AI AGENT FOR TICKET CREATION

Users must open tickets through conversation.

Create an AI chat agent.

User interaction example:

User:
"Meu notebook não liga depois da atualização."

Agent:
"Entendi. Vou registrar um chamado para suporte de hardware.
Você poderia informar qual modelo do notebook?"

Conversation continues until ticket is ready.

Agent responsibilities:

1 understand problem
2 extract structured information
3 create ticket automatically

Fields extracted:

ticket_subject
ticket_description
product_purchased
ticket_channel

Then system calls:

POST /ticket/create
